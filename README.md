# 屏蔽反诈模块 (Anti-Fraud Block)

Magisk 模块，屏蔽 ColorOS 系统内置反诈相关组件 + 屏蔽国家反诈 App 服务及反诈相关 200+ 反诈 IP。

## 功能
- 屏蔽 ColorOS 反诈系统服务
- 屏蔽国家反诈中心 App 相关服务
- 屏蔽 200+ 反诈相关 IP
- iptables 自定义规则
- 内核级拦截

## 安装
1. 下载 zip 并在 Magisk 中刷入
2. 重启设备
