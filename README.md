# 屏蔽反诈模块 (Anti-Fraud Block)

Magisk / KernelSU 模块，用于屏蔽 Android 设备上的反诈相关组件。

## 版本

- **v3** (2026-05-17)

## 安装

1. 下载 `block_anti_fraud_v3.zip`
2. 在 Magisk/KernelSU 中刷入
3. 重启设备

## 功能

- 屏蔽反诈相关系统服务
- 自定义 iptables 规则
- 内核级拦截
